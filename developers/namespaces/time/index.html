time

The time namespace contains functions that offer more precise timing related information. Lua only offers two time related methods by default which don’t offer much in regards to precision timing, or other useful time related info.

Functions exposed by this namespace are accessed via the prefix: hook.time.

Functions

clock

Returns the current high-resolution clock information. (Uses std::chrono::high_resolution_clock::now)

1table hook.time.clock();
Parameter NameTypeDescription
None.
Returns
(table) Table containing the high-resolution clock information.

The returned table contains the following properties:

  • s (number) - The number of seconds of the high-res clock.
  • ms (number) - The number of milli-seconds of the high-res clock.
  • micro (number) - The number of micro-seconds of the high-res clock.
  • nano (number) - The number of nano-seconds of the high-res clock.
